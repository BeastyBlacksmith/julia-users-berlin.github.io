Julia Users Group - Berlin

Hi all, we started a basic Julia get together in 2015. It was nice to get to meet some of the other Julia users here in Berlin.

Following the recent PyData conference in Berlin a lot of interest was expressed in meeting more regularly to discuss Julia related topics. I have set up this GitHub webpage in order to aid with co-ordination and so that there will be a landing page when people search for Julia in Berlin.

The discussion organising the first meetup is on the Julia Users google groups forum here
